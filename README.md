# Event Ticketing Platform

An End-to-End Event Lifecycle & Ticketing Platform built with Node.js, Express, and MySQL.

## 🚀 One-Click Deploy

I've set up the configuration below so you can deploy this project instantly. Just click the button of your choice:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/sandeep21-07-2004/event-ticket-platfrom)

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/deploy?template=https://github.com/sandeep21-07-2004/event-ticket-platfrom)

## 🛠️ How to Finish the Deployment

Since I don't have your passwords, you just need to do this:

1.  **Click the "Deploy to Render" button above.**
2.  **Log in to GitHub** when it asks you.
3.  **For Database**: It will ask for a `DATABASE_URL`. You can get a free one at [TiDB Cloud](https://tidbcloud.com/) or just use the Railway button instead (which creates the database for you!).

## Features
- User Registration & Authentication (JWT)
- Event Creation & Management
- Ticket Generation
